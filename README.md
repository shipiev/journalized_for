# JournalizedFor

Plug-in for Redmine. It journalize has_one and has_many association

## Installation

Add this line to your application's Gemfile:

    gem 'journalized_for'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install journalized_for
